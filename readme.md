# 📌 Overview

This project presents an interactive **Supply Chain & Logistics Dashboard** built in **Power BI** using multiple CSV datasets. The dashboard helps monitor supplier performance, inventory levels, production, shipments, customer sales, and overall business performance through interactive KPIs and visualizations.

---

# 🎯 Business Objectives

- Monitor supplier performance and lead times.
- Track inventory availability and stock movement.
- Analyze production efficiency and defect rates.
- Evaluate shipment delays and logistics performance.
- Measure customer sales and revenue trends.
- Support data-driven supply chain decisions.

---
